## 🚀 **New Usage Patterns**

### **For Users:**
```bash
run.bat                                    # Windows menu launcher
python finance_tracker/main.py            # Modern UI (default)
python finance_tracker/main.py --ui classic # Classic UI
python ui_demo.py                         # UI selection demo
```

### **For Developers:**
```bash
python build.py                           # Build everything
python build.py --icon-only              # Just create icon
python build.py --exe-only               # Just build executable
python build.py --help                   # Show all options
```

### **For Distribution:**
```bash
build_for_sharing.bat                     # One-click build for Windows
python build.py                          # Cross-platform build
```
