## 🛠️ Troubleshooting for Friends

### "Windows protected your PC" message
- Click "More info" → "Run anyway"
- This is normal for new apps

### App won't start
- Run as Administrator
- Check Windows Defender hasn't quarantined it
- Make sure they have Windows 10+

### Missing features
- Make sure they extracted the complete ZIP
- Try running install.bat again
